# Get started
## Requirements


For running DreamNetwork Hypervisor on your own, you'll need a little stuff.
- A server running Linux (Debian/Ubuntu/CentOS) with suitable performances <br/> (Minimum 4Gb of RAM, 2 vCores and 30Go storage (SSD/NVMe recommended).<br/>

- SSH access to this server
- Some basics skill of Linux systems
