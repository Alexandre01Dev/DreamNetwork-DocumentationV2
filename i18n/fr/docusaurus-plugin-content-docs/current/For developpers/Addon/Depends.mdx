# Les d√©pendences


Ajoutez DreamNetwork-api.jar dans un dossier facilement accessible par votre IDE, dans le cas de l'exemple ci-dessous, il se situra dans le dossier libs.


Rajouter dans votre pom.xml

## Maven
```xml
 <repositories>
            <repository>
                <id>dreamnetwork</id>
                <url>https://repo.dreamnetwork.cloud/repository/maven-snapshots/</url>
            </repository>
        </repositories>
```

```xml
    <dependency>
        <groupId>be.alexandre01.dreamnetwork</groupId>
        <artifactId>Core-API</artifactId>
        <version>1.11.0-SNAPSHOT</version>
        <scope>system</scope>
    </dependency>
    <dependency>
        <groupId>org.projectlombok</groupId>
        <artifactId>lombok</artifactId>
        <version>1.18.22</version>
        <scope>compile</scope>
    </dependency>
```

## Gradle :

```java
repositories {
    maven {
        url "https://repo.dreamnetwork.cloud/repository/maven-snapshots/"
    }
}
```
```java
    implementation 'be.alexandre01.dreamnetwork:Core-API:1.11.0-SNAPSHOT'
    implementation 'org.projectlombok:lombok:1.18.22'
```
