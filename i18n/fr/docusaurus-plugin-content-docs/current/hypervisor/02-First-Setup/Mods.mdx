# STATIC ET DYNAMIC ⚡
> ** What are the differences between STATIC mode and DYNAMIC mode ?**

#### STATIC : 

Ce mode permet de démarrer le serveur à partir du serveur modèle ("template") que vous avez créé. Le serveur ne peut pas être démarré plusieurs fois alors qu'il est en marche

#### DYNAMIC : 
Ce mode copie le serveur dans un dossier temporaire et le lance à partir de celui-ci. Le serveur peut être lancé plusieurs fois. Lorsque le serveur est fermé, le serveur copié est supprimé. C'est utile pour les serveurs de mini-jeux/lobbies.

:::caution
Le mode dynamique est actuellement en cours de développement et peut être modifié en raison de l'optimisation.
Pour optimiser le réseau, veuillez ne pas avoir de dossier de grande taille dans votre modèle pour le mode dynamique.
:::
