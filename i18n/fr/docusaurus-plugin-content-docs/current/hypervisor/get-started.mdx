# Pour commencer
## Conditions requises


Pour exécuter DreamNetwork Hypervisor sur votre propre, vous aurez besoin de quelques trucs.
- Un serveur fonctionnant sous Linux (Debian/Ubuntu/CentOS) avec des performances adaptées <br/>(Minimum 4Gb de RAM, 2 vCores et 20Go de stockage (SSD/NVMe recommandé).<br/>

- Accès SSH à ce serveur
- Quelques connaissances de base des systèmes Linux
